🌿 Branch & Commit Standards

Branch Naming - Include Issue ID

Keep your existing naming convention, just ensure issue ID is included:

your-feature-branch-123 (existing + issue ID)
bugfix-login-error-456 (existing + issue ID)
hotfix-security-patch-789 (existing + issue ID)

Commit Messages - Add Issue References

We already have good commit formats, just add issue links:

feat(auth): add OAuth2 support (closes #123)
fix(api): resolve null pointer (fixes #456)
docs: update installation guide (refs #789)
